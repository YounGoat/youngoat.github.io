Electron

Electron

Build cross platform desktop apps with JavaScript, HTML, and CSS

快速入门

安装

通过 NPM 安装 Electron 运行时环境是较简单的安装方式，但由于需要编译的关系，这种安装方式速度较慢： 

npm install -g electron

也可以直接下载与当前平台匹配的已编译版本。国内可以通过淘宝 NPM 镜像下载： 

https://npm.taobao.org/mirrors/electron

编写

（略）

运行

electron myapp

分发

将源代码与 Electron 运行时环境相结合，就成为一个可独立分发的新应用。以 Mac OS X 为例： 

# 将源代码复制到特定目录（该目录默认是不存在的）。
mv myapp <Electron.app>/Contents/Resources/app

# 现在 <Electron.app> 即可以作为一个独立分发的新应用。

打包

# 安装打包工具。
npm install -g asar

# 打包。
asar pack myapp myapp.asar

# 运行（仍依赖于运行时环境）。
electron myapp.asar
