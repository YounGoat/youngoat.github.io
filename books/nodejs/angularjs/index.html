AngularJS

AngularJS

如今的大型开源项目，凡有生命力者，其文档大都已自成体系，且会跟随项目版本的更迭而不断演进。对于缺乏经验的开发者而言，如何掌握阅读文档的正确姿势，即已构成挑战。有的文档会提供详细的阅读指引，对读者较好友好；然而章法隐晦、灰蛇草线者有之，天马行空、无拘无束者亦有之。所幸，AngularJS 的文档大抵属于前者，如果你打算跟随官方文档学习，强烈建议先读一下其概览：

DOCUMENTATION OVERVIEW
https://angular.io/docs/ts/latest/guide/

How to read and use this documentation

快速入门

  • QUICKSTART
    https://angular.io/docs/ts/latest/quickstart.html

  • LEARNING ANGULAR
    https://angular.io/docs/ts/latest/guide/learning-angular.html

两者都是官方文档的组成部分，面向初次接触 AngularJS 的开发者。QUICKSTART 提供了免安装的开胃小菜，真正的入门指引，应当从 LEARNING ANGULAR 开始。但不论走哪条途径，都要求开发者拥有相当的前端开发基础。

  1. 下载
    QuickStart seed 是官方提供的快速入门样例。
