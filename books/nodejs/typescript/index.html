TypeScript

TypeScript

快速入门

  1. 准备素材

    TypeScript starts from the same syntax and semantics that millions of JavaScript developers know today.

    TypeScript 是 JavaScript 的超集，也就是说，它完全兼容后者的语法和语义。因此，我们准备的素材可以是一段最简单的 JS 代码，我们赋予代码文件 .ts 后缀名，代表这是一个 TypeScript 程序：

    /* hello.ts */
function main() {
  console.log('Hello world!');
}

main();

  2. 安装工具

    # 安装命令行编译工具，创建全局命令 tsc
npm install -g typescript

# * 也可以根据自己的开发工具（比如 Atom），安装相应的支持组件
apm install atom-typescript

# 执行编译
tsc hello.ts
# 输出：将创建同名 .js 文件
