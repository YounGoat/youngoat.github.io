Start With Starter Kit

Starter Kit 最初是 ReactJS 官方随较早的版本提供的开箱即用的应用代码，后来已演变为一套复杂的开发工具，并拥有独立的官方网站：
https://www.reactstarterkit.com

ReactJS 15.0.2

  1. 下载

    我们可以从 GitHub 下载 Starter Kit：
    https://facebook.github.io/react/downloads/react-15.0.2.zip

  2. 启动

    下载完成后，需要将其装入一个本地 Web 容器（比如 Nginx）中，以使我们能够通过 HTTP 方式访问。

  3. 依赖资源本地化

    # 在项目根目录下安装 babel 模块。
cd react-15.0.2
npm install babel

    <!-- 修改 index.html-->
<!-- 用本地脚本取代 CDN 服务 -->
<!-- <script src="https://cdnjs.cloudflare.com/ajax/libs/babel-core/5.8.23/browser.min.js"></script> -->
<script src="/bower_components/babel/browser.js" charset="utf-8"></script>
