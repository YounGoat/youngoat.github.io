ReactJS 官方指导是入门的另一佳径。

Getting Started

# 下载项目源代码
git clone https://github.com/reactjs/react-tutorial

# 启动服务
cd react-tutorial
node server.js

如果需要完全离线运行，请先将项目中的依赖资源本地化：

# 安装依赖模块
cd react-tutorial
npm install react
npm install babel
npm install jquery
npm install marked

# 修改 react-tutorial/public/index.html
# 用本地脚本替换 cdnjs.cloudflare.com 服务

# 重启服务，OK!
node server.js

接下来，依 Tutorial 指示按部就班。
