Java 简史

Java 2

到目前（2017年1月）为止，Java 发布的 9 个主要版本，一直沿用 1.x 的版本号。习惯上，我们省略主版本号，直接使用次版本号来称呼相应版本的 Java，比如用“Java 8”指代 Java 1.8.x，惟独 Java 2 是个特例。自 1.2 开始的所有版本都可被称为 Java 2，它几乎成为 Java 的标准名称，包括 J2SE / J2EE / J2SDK 等术语沿用至今。
