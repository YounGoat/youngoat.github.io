Mac OS X 中的 Java

在哪里？

不同的安装方式会使用不同的默认安装路径，可以尝试打开以下目录：

# JRE
/System/Library/Frameworks/JavaVM.framework/

# JDK
/Library/Java/JavaVirtualMachines/
